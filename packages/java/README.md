# liter-llm - Java Bindings

Universal LLM API client — 142+ providers, streaming, tool calling. Rust-powered with polyglot bindings.

## Installation

Add to your `pom.xml`:

```xml
<dependency>
    <groupId>dev.kreuzberg.literllm</groupId>
    <artifactId>liter-llm</artifactId>
    <version>1.2.2</version>
</dependency>
```
