# liter-llm - Elixir Bindings

Universal LLM API client — 142+ providers, streaming, tool calling. Rust-powered with polyglot bindings.

## Installation

Add `:liter_llm` to your `mix.exs` dependencies:

```elixir
defp deps do
  [
    {:liter_llm, "~> 1.2.2"}
  ]
end
```
